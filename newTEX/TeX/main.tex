\section{Сборочный чертеж устройства }
Разработанное устройство состоит из четырех основных частей. Первая часть устройства - оптическая скамья, она является отдельным элементом. Основной частью является зеркало, расположенное на оптической скамье, угол которого исследует устройство с ФПУ. Также в устройстве есть лазер и плата с ФПУ для измерения малых угловых перемещений. Все устройство получает питание через IEС. Выходные данные можно получить через порт D-SUB. Порты располагаются на лицевой части устройства. Внутри корпуса устройства определения малых угловых перемещений сразу после порта IEС находится трансформатор и печатная плата.\par
Сборочный чертеж устройства для измерения малых угловых перемещений представлен на листе КСУИ.256.Р3340.003 СБ в конце проекта. На сборочном чертеже представлена оптическая скамья, на которой расположены элементы устройства и объект измерений (зеркало). Все элемента закреплены на креплениях оптической скамьи с помошью эпоксидного клея.
