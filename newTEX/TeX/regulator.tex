\begin{itemize}

По виду переходного процесса можно сделать вывод, что модель пьезоактуатора содержит две составляющие – апериодическую и колебательную. 
Из формулы \ref{eq:l=2x} следует что:
\begin{gather*}
	L = 2 * 5*10^{-6} = 10 * 10^{-6},\\
	\alpha = \frac{L}{r} = 0.002 \text{ мрад } = 1'. 
\end{gather*}
где
\begin{ESKDexplanation}
	\item $\alpha$ - угол поворота ротора при одном шаге.
\end{ESKDexplanation}
Полученное значение полностью удовлетворяет поставленному в техническом задании, а также повторяет заявленное значение шага в технической характеристике.

\subsection{Синтез регулятора пьезодвигателя}

Для нивелирования колебаний в системе необходимо синтезировать ПИД регулятор.
Воспользуемся методом Циглера-Никольса для синтеза ПИД регулятора.

Из реакции системы на ступенчатое изменение управляющего сигнала,
данная характеристика называется кривой разгона, можно найти коэффициенты ПИД регулятора обычно они равны:
\begin{gather*}
	K_п = 0.6k^*_п,\\
	К_и = 1.2\frac{k^*_п}{T^*},\\
	K_д = 0.075\frac{k^*_п}{T^*}.
\end{gather*}
где
\begin{ESKDexplanation}
	\item $k^*_п$ - значение коэффициента передачи регулятора, при котором система находится на границе устойчивости.
\end{ESKDexplanation}
В данном случае использование только интегрального коэффициента полностью выполняет требования к регулятору.
\begin{gather*}
	K_п = 0,\\
	K_и = 1168.59747755308,\\
	K_д = 0.
\end{gather*}

Построим передаточную функцию по данным коэффициентам
\begin{gather}
	W_p(s) = K_п + \frac{K_и}{s} + \frac{K_д*s}{T_д*s+1},\\
	W_p(s) = \frac{1168.59747755308}{s}.
\end{gather}
где
\begin{ESKDexplanation}
	\item $T_д$ - постоянная времени, характеризующая инерционные свойства
	реального дифференцирующего звена. 
\end{ESKDexplanation}
